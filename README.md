## 프로젝트 소개
저희 프로젝트는 경제에 대한 관심이 높아지고 있는 현 상황에서, 주식 시장에 뛰어들고 있는 2030세대를 대상으로 안전하고 신뢰할 수 있는 주식 투자를 할 수 있도록 지원하기 위해 개발된 주식 예측 모델을 소개합니다. 이 모델은 데이터 분석을 통해 주식 시장의 동향을 예측하고, 효과적인 투자 전략을 제공합니다.
